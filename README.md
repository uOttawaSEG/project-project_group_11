# OTAMS - Online Tutoring Appointment Management System
Course: SEG2105 (Fall 2025)

## Overview
OTAMS is a mobile application designed to streamline the process of tutoring
appointment scheduling at the University of Ottawa Help Centre.

## Features
- Students and Tutors can login and logout
- All info stored in a database
